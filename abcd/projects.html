1. Static Website Hosting on AWS S3

I hosted a fully functional static website on AWS S3 using a custom HTML, CSS, and images. I configured public access, uploaded files, and tested the website URL. This project helped me understand cloud storage, hosting, and permissions in AWS.

2. Serverless Image Upload Application

Built a serverless application where users can upload images to S3, and it triggers a AWS Lambda function. The Lambda processes the image and sends a notification via AWS SNS. This project gave me hands-on experience with event-driven architecture, serverless computing, and cloud automation.

3. IAM User & Role Management Project

Practiced creating IAM users, roles, and policies in AWS. I simulated real-world scenarios: granting least-privilege access, creating groups, and managing permissions for different projects. This improved my understanding of cloud security, authentication, and authorization.

4. Python Automation Scripts for AWS

Developed Python scripts to automate common AWS tasks such as creating S3 buckets, uploading objects, and managing resources. Learned to integrate Boto3 library with Python and make cloud operations efficient and reproducible.

5. Cloud Project: Portfolio Website

Created my personal portfolio website hosted entirely on AWS, linking multiple pages like Home, Early Life, Education, Career, Projects, and Timeline. This project gave me practical experience in website hosting, structuring projects, cloud deployment, and user experience design.

Project Summary

Through these projects, I strengthened my skills in AWS cloud services, Python programming, serverless computing, and web development. Each project represents a real-world scenario and helped me become more confident in solving problems using cloud technologies.
